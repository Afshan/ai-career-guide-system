# AI-Driven Student Performance and Career Guide System

An intelligent web-based platform designed to help students track academic performance, manage goals, and receive AI-powered career guidance based on their interests, academic data, and progress.

---

# 📌 Project Overview

The **AI-Driven Student Performance and Career Guide System** is a Final Year Project developed to assist students in analyzing their academic journey and discovering the most suitable career path using AI-driven recommendations.

The system helps students:
- Track academic performance
- Manage subjects and marks
- Monitor goals and productivity
- Visualize progress using charts and analytics
- Receive AI-based career suggestions
- Generate personalized learning roadmaps

This platform focuses on improving student productivity, career planning, and self-development through data-driven insights.

---

# ✨ Features

## 🔐 Authentication System
- User Registration & Login
- Session Handling
- Secure Logout Functionality

---

## 🎯 Interest Selection
During registration, students can select their interests, which are stored in the database and later used for AI-based career recommendations.

---

## 📚 Academic Information Management
Students can:
- Add Degree Information
- Enter Semester Details
- Add CGPA
- Save Academic Records

---

## 📊 Subject & Marks Tracking
- Add Subjects and Marks
- Store Subject Data in Database
- Visualize Academic Performance
- Dashboard Graphs & Analytics

---

## 🤖 AI Career Suggestion System
The system analyzes:
- Interests
- Academic Performance
- Goals
- Degree Information
- Student Progress

Based on this data, the AI suggests the most suitable career path for students.

---

## 🛣️ AI Career Roadmap
The platform generates a step-by-step learning roadmap for students.

Example:
If a student wants to become a Web Developer, the system provides:
- Required Skills
- Technologies to Learn
- Learning Sequence
- Suggested Learning Path

Google Gemini API integration is planned for enhancing AI-generated recommendations and roadmap generation.

---

## 🎯 Goal Tracking System
Students can:
- Add Goals

- Track Goal Completion
- Monitor Goal Progress

AJAX-based interactions are used for smooth updates without page reloads.

## 📉 Progress Analytics

## ⚙️ Settings Management

# 🗂️ System Workflow

1. User lands on the website landing page
2. User registers using email and password
3. User selects interests
4. Academic information is added
5. Student is redirected to dashboard
6. New students are prompted to add subjects and marks
7. Subject data is stored in the database
8. Dashboard visualizes progress through charts
9. AI suggests suitable careers
10. Students manage goals, subtasks, and to-do lists
11. Progress analytics help monitor academic performance over time
